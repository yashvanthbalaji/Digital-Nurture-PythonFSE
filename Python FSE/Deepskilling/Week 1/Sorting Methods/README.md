## 🫧 Bubble Sort

Bubble Sort repeatedly compares adjacent elements and swaps them if they are in the wrong order until the list is sorted.

![Bubble Sort Output](screenshots/bubble-sort-output.png)

---

## 🎯 Selection Sort

Selection Sort finds the smallest element in the unsorted part of the list and places it at the correct position.

### Output

![Selection Sort Output](screenshots/selection-sort-output.png)

---

## 📥 Insertion Sort

Insertion Sort inserts each element into its correct position in the already sorted part of the list.

### Output

![Insertion Sort Output](screenshots/insertion-sort-output.png)

---

## ⚡ Quick Sort

Quick Sort uses a **random pivot** to divide the list into smaller and larger elements and recursively sorts both parts.
